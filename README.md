# Калкулатор - курсова работа

Описание:
Този проект представлява прост калкулатор, написан на Python.
Калкулаторът поддържа следните операции:
- Събиране
- Изваждане
- Умножение
- Деление

# Структура на проекта:
- models.py - съдържа класа Calculator с методите за изчисление
- main.py - главният файл, който стартира програмата с меню
- requirements.txt - списък с използвани библиотеки
- readme.txt - този файл

Използвани библиотеки:
Проектът не използва външни библиотеки. Използват се само вградените
функции на Python, затова файлът requirements.txt е празен.

# Как се използва:
1. Стартирайте програмата с командата: python main.py
2. Изберете операция от менюто (1-5)
3. Въведете двете числа
4. Програмата ще покаже резултата
5. За изход изберете опция 5
